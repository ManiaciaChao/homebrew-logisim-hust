## Install

```sh
brew tap colinaaa/logisim-hust
```

Then:

```sh
brew cask install logisim-hust
```

Also offer the Cask for [logisim-ita](https://github.com/LogisimIt/Logisim),
install with:

```sh
brew cask install logisim-ita
```
